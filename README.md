# Personal Knowledge Vault (Terminal)

A simple terminal-based notes app using TXT storage.

## Files
- vault.py — main Python program (terminal interface)
- notes.txt — sample data (example entries)
- README.md — this file

## Requirements
- Python 3.8+ (installed on your machine)

## How to run (locally)
1. Download the repo as ZIP (Code → Download ZIP) OR click the green *Code* button → *Download ZIP*.
2. Unzip and open a terminal in the folder.
3. Run: python vault.py
4. Use the menu to create/view/search/edit/delete notes. Search uses keywords (case-insensitive) and you will select which matching note to act on.

Note format :
TITLE:<title>
BODY:<body>
